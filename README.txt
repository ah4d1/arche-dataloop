This is Arche-Dataloop software.
********************************
Arche-Dataloop is a software for data scientists.
The project was started in 20th October, 2017.

The components :
********************************
Codes are written in Pascal via Lazarus IDE.
We use a lot of R function to perform data analytics, data mining, and statistical modeling.

The versions.
********************************
Every single update comes with a version number like 0.0.3-D or 0.0.4-R.
D character means development version.
R character means release version.
For your convenience, please always use the software with R character.

Installation
********************************
For installation in Windows, please download Arche-Dataloop.win.x.x.x-R.zip.
For installation in Linux, please download Arche-Dataloop.linux.x.x.x-R.zip (not available yet).
And then just extract at specific directory and run the binary file.
x.x.x-R refers to the release version number.

Dependencies
********************************
Please install R before using this app and then set RScript binary file through Tools > Settings menu.
R can be found at https://www.r-project.org/
